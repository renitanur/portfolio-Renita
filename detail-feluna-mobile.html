Feluna - Aplikasi Pelacak Menstruasi (Mobile)

Proyek Aplikasi Flutter | Tahun 2024

Deskripsi Proyek

Feluna adalah aplikasi mobile yang dirancang untuk membantu wanita dalam memantau siklus menstruasi secara efisien dan intuitif. Aplikasi ini tidak hanya menampilkan kalender siklus, tetapi juga menyertakan fitur-fitur pendukung seperti latihan pernapasan dan chatbot konsultasi.

Fitur Utama

  • Calender : Pelacakan siklus menstruasi secara visual dan interaktif
  • Treatments : Latihan pernapasan untuk relaksasi saat PMS
  • Chatbot : Chatbot cerdas untuk konsultasi seputar kesehatan reproduksi

Teknologi & Tools

  • Figma untuk desain UI/UX
  • Flutter & Dart untuk pengembangan UI mobile
  • Firebase untuk otentikasi dan database
